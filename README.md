# Workforce Analytics System

## Overview

A workforce analytics project developed using Python, Pandas, NumPy, and Matplotlib to analyze employee attendance, attrition trends, departmental performance, and salary distribution.

## Features

* Employee Attendance Analysis
* Attrition Rate Analysis
* Department Performance Evaluation
* Salary Distribution Analysis
* Automated Data Visualization
* Report Generation

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib

## Project Structure

```text
workforce_analytics/
│
├── attendance_analysis.py
├── attrition_analysis.py
├── department_analysis.py
├── salary_analysis.py
├── visualization.py
├── main.py
├── employee_data.csv
├── requirements.txt
└── reports/
```

## Run Project

```bash
pip install -r requirements.txt
python main.py
```

## Output

The system generates:

* Department Performance Chart
* Salary Distribution Chart
* Statistical Workforce Reports
